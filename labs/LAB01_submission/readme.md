# Interstate Zero

## Story
A life lived by ignoring others to chase meaningless pursuits has caught up to you. You find yourself driving into Interstate Zero, a world which reminds you of moments in the past where you made the wrong choices. Nothing here is the way it should be. Search for the reason you were brought here, and confront the damage you have caused - then you might be permitted to return to your life. But be careful - your limited fuel and fragile mind are targets for the things that roam these roads. Deplete either, and you'll find yourself stranded with these creatures forever.

## How to Play
The gameplay itself is rather simple. Use text input to explore the locations in front of you, gather clues, and handle encounters with entities. Be careful to manage your fuel and mental stability levels. Read the text carefully, as there might be subtle clues which could assist you.

### Running the Game
```bash
python game.py
```
Run the game in a separate, fullscreen terminal window if you can! Not that it won't work, but it makes a huge difference for immersion. Plus, my ASCII title card doesn't fit in the VS Code terminal.

### Commands
When exploring a location, type 1, 2, or 3 to explore the corresponding area, or L to leave.
When leaving a location and choosing a direction to travel, type the direction.

## Goal
Collect each of six clues, then find the exit, without running out of fuel or mental stability.

## Tips
Again, be sure to read the text carefully. The options given to you during an encounter have names which reflect the consequences you should expect when choosing them. I'd also recommend trying to find certain clues before you encounter certain entities... but that requires a little luck.
